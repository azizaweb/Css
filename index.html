Cascading Style Sheets CSS

Cascading Style Sheet Css veb-sahifa yaratuvchilari tomonidan ranglar, shiftlar, alohida bloklarning taribini va ushubu veb-sahi bo'lishining boshqa jihatlarini belgilash achun ishlaţjladi. Css-ni ishlab chiqishning asosiy maqsadi veb-sahifaning (HTML yoki boshqa belgilash tillari yordamida ishlab chiqarilgan) mantiqiy tuzilishining tavsifini ushbu veb-sahifa (hozir rasmiy Css tili yordamida ishlab. chiqarilgan) tashqi ko`rinishi tavsifidan ajratish edi. Ushbu ajratish hujatning mavjudligini oshirishi, ko'proq mosłashuvchanligi va taqdimotni boshqarish qobiliyatini ta' minlab, shuningdek tarkibiy tarkibdagi murakkablik va takrorlanuvchanlikni kamaytirishi mumkin.
